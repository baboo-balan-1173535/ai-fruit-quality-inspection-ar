Setup

The glasses app finds this server automatically on any WiFi (UDP discovery, no IP typing). Checklist if it can't connect:

1. Phone and laptop on the same WiFi, phone VPN off.
2. Run setup_admin.bat once (firewall + database).
3. AR switch above is ON.
4. Older APK builds (before auto-discovery) need the IP above typed into the Unity Inspector and a rebuild.
